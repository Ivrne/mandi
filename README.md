Jamie Coffee
============

This is the codebase for Jamie Coffee App (API & Client)

Requirements
------------

- Node.js (v7.0.0+)
- MongoDB (v3.2.0+)

Set-up
-----

To set-up this workshape.io codebase on your development environment please follow these steps:

```bash
git clone https://www.github.com/tancredi/jamie
cd jamie
npm install
npm run fixtures
npm run dev
```
